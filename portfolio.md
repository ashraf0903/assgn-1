# My First Web Development Project

This is a simple profile page created for my first web development assignment.

## Project Overview
This project displays a basic profile card with my name, a short bio, and links to my work.

## Tools Used
*   **HTML**: To create the structure of the page.
*   **CSS**: To add colors, fonts, and center the content.
*   **JavaScript**: To add a simple click alert to the button.
